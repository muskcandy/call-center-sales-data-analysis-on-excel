# call-center-sales-data-analysis-on-excel
📌 Project Overview

This project presents an interactive Excel dashboard for Call Center Data Analysis (2023). The objective is to transform raw operational data into actionable business insights using Excel-based analytics, visualizations, and KPIs. The dashboard enables stakeholders to monitor call performance, agent productivity, customer behavior, revenue trends, and service quality in a single unified view.

The project demonstrates practical data analyst skills, including data cleaning, KPI formulation, pivot analysis, visualization design, and dashboard storytelling.

🎯 Objectives

Analyze call volume trends across months and weekdays.

Evaluate sales performance by representatives and cities.

Monitor customer satisfaction using ratings.

Track operational KPIs such as total calls, revenue, call duration, and happy callers.

Compare male vs female caller distribution across cities.

📈 Dashboard Insights
1. Monthly Call Trend

Highest call volume observed in October.

Lowest activity recorded in December.

Mid-year dip during July–August, indicating possible seasonality.

2. City-wise Caller Distribution (Male vs Female)

Cleveland shows the highest total callers.

Female callers dominate in Cleveland, while Columbus has more male callers.

Provides actionable insight for targeted customer engagement strategies.

3. Weekly Call Volume

Wednesday records the highest number of calls.

Saturday has the lowest volume, indicating lighter weekend demand.

Useful for workforce scheduling and shift planning.

4. Sales Performance by Representatives

Top-performing representative: R03.

Consistent high performers: R02 and R05.

Enables performance benchmarking and incentive planning.

5. Customer Ratings Analysis

Majority of customers provided 4 and 5-star ratings, reflecting good service quality.

Very few low ratings, indicating strong operational efficiency.

6. Top 5 Callers Sales Analysis

High-value customers identified for each city.

Helps in customer retention strategies and loyalty programs.
